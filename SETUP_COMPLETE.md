## 🎨 Customization Tips

### Change Colors
Edit `web/src/theme.js`:
```javascript
primary: {
  main: '#YOUR_COLOR',
}
```

### Update Logo
Replace "RMS" in `Navbar.jsx` with your logo image

Edit `HomePage.jsx` to change:
- Hero text
- Features
- Menu items
- Testimonials
- Contact info
